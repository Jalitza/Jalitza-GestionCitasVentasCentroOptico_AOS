# Jalitza-GestionCitasVentasCentroOptico_AOS
README.md

📌 Nombre del Proyecto

Gestión de Citas y Ventas en un Centro Óptico

📖 Introducción

Este proyecto tiene como objetivo optimizar la gestión de citas y ventas en un centro óptico mediante una plataforma web. La solución permite a los clientes agendar citas, consultar disponibilidad y gestionar compras de productos ópticos de manera eficiente.

🎯 Objetivo y Alcance

Objetivo:

Facilitar la administración de citas y ventas para centros ópticos.

Automatizar la gestión de clientes, productos y servicios.

Mejorar la comunicación entre clientes y empleados.

Alcance:

Módulo de gestión de citas.

Módulo de ventas y facturación.

Administración de clientes y empleados.

Reportes y estadísticas.

Integración con Firebase para autenticación y almacenamiento de datos.

🔍 Problemática

Los centros ópticos suelen enfrentar dificultades en la organización de citas y en la gestión de ventas, lo que puede generar pérdidas de clientes y disminución en la eficiencia operativa. Este proyecto busca solucionar estos problemas mediante una plataforma digital intuitiva y de fácil acceso.

🌍 Impacto Esperado

Reducción en el tiempo de administración de citas.

Mejora en la experiencia del cliente.

Aumento en la eficiencia operativa.

Digitalización y optimización de procesos de ventas.

🛠️ Tecnologías Utilizadas

Frontend: Angular + Tailwind CSS

Backend: Spring Boot + PostgreSQL

Servicios adicionales: Firebase (autenticación, almacenamiento de datos)

Infraestructura: GitHub + Jira

📌 Gestión del Proyecto

Herramientas: Jira para la gestión de tareas y GitHub para el control de versiones.

Metodología: Agile/Scrum

👥 Roles del Equipo

Desarrollador Frontend: Implementación de la interfaz de usuario.

Desarrollador Backend: Desarrollo de la API y la lógica de negocio.

Gestor del Proyecto: Supervisión del desarrollo y planificación.
